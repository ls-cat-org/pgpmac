258 # motor_dict keys
259 # motor_num: The pmac motor number between 1 and 32 inclusive. Leave undefined or set to -1 for motor for DAC and Binary Output motor like objects
260 # coord_num: The coordinate system the said motor finds itself in between 1 and 16 inclusive. Leave undefined or 0 for DAC and Binary Output motor like objects.
261 # max_accel: counts/msec/msec
262 # max_speed: counts/msec
263 # u2c: The conversion between counts and user units: Multiply user units by u2c to get counts. Should never be zero.
264 # active: 1 if the motor should be set up and used, 0 otherwise
265 # hard_ini: The section name for this motor in the microdiff_hard.ini file
266 # moveMode: freeRotation, rotation, or translation (default) used for the LS-CAT GUI
267 # reference: (omega only) The angle for which centering.y is up and centering.x is positive downstream
268 # axis: The axis letter for the PMAC in the specified coordinate system (X, Y, Z, etc)
269 # neutralPosition: The offset in user units between the home position and what we want to call zero
270 # printf: The printf format string for the position in the ncurses interface (uses a field width specifier *)
271 # format: The printf format string to update the redis value
272 # maxPosition: The software upper limit in user units relative to the home position
273 # minPosition: The software lower limit in user units relative to the home position
274 # smallStep: Recommened small step value for a user interface
275 # largeStep: Recommened large step value for a user interface
276 # update_resolution: Don't update redis until the position has changed by this amount in user units
277 #
278 # NOTE: active_init, home, and inactive_init should only be specified if the default string will not serve the purposes such as
279 # for omega and the polarizer
280 #
281 # active_init: A comma separated list of strings (double quoted if spaces present) enclosed in braces to send to the PMAC when the motor is active.
282 # home:` A comma separated list of strings (double quoted if spaces present) enclosed in braces to send to the PMAC to home the motor
283 # inactive_init: A comma separated list of strings (double quoted if spaces present) enclosed in braces to send to the PMAC when the motor is inactive.
